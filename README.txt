Casino Night

This game is a 4 in 1 casino game, housing Blackjack, Craps, Roulette, Slots. users will start
the game with $1000, and have the option to be $5, $10, $25, $50, $100, $250, $500 on each game
except slots which is hard betting $5. Users will be able to view their last 10 games, which
will display the winnings/ loosing and the game that they were playing.

Prerequisites

Minimum resolution 1400 X 900 
Proper IO devices
Java installed on machine

Installing

Extract contents of .zip file
Double click .jar file
Game will open

Logging in

If the user wants to create a user they will click the new user button then input their desired username

If the user want to use an existing user they will click the returning user button and input their username

Losing all your money

Upon the user hitting $0 they will be sent back to the main menu and have thier funds reset to $1000
